# Market-Partitioning Project

The retailer has brought us in to segment their customer base using transaction data rather than assumptions. They've shared a purchase history at transaction level, and the task is to build a clustering model that identifies which customer groups actually exist in the data.
The model needs to work across two levels: overall spending patterns and what customers buy at the product level. Each segment should map to a real commercial difference, not just a statistical one.
